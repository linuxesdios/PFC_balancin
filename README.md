#PFC_balancin
============
# Para la realizacion de el proyecto final de carrera se creo un balancin controlado por imu y dspic

La realizacion es mediante 
 - IMU mpu6050
 - Dspic30f4013 
 - dos trassitores td130 
 - 1 bluetooth hc-05
 - 2 botones
 - 6 led
